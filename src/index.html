Passphrase recovery tool

This tool helps you recover misspelled passphrases. For each word it will detect if it is in the chosen wordlist, and suggest similar ones. You can use this to generate a list of passphrases that are close to the one provided. All of this works offline in your browser with no servers. You should be able to download save this page (Ctrl + S) and use it safelly without internet. Grab the source code from the github page.
